About This Project

This project is a secure temporary file sharing system developed as part of a university academic submission. Our goal is frictionless security.

  • No login or registration required
  • Unique, auto-generated access codes
  • Time-limited downloads for privacy
  • Secure file handling via modern web tech

Powered by Node.js, Express, SQLite, and beautiful web technologies.
